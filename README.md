# Urban Aura Apparel Website

A simple static clothing brand website with:

- Homepage with featured collection and brand story
- Products page with add-to-cart actions
- About page with mission and customer testimonials
- Contact page with a demo form
- Cart page with local storage support

## Run locally

Open `index.html` in your browser, or use a local file server for best results.

### Optional local server

From the project root:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.
